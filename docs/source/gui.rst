DaoLITE GUI
===========

Overview
--------

The ``daolite.gui`` module provides graphical tools for designing and visualizing Adaptive Optics (AO) pipelines in DaoLITE. It is intended for users who prefer a visual interface to build, edit, and inspect AO processing chains.

Features
--------
- Visual pipeline designer for AO systems
- Drag-and-drop component creation and connection
- Real-time visualization of pipeline structure and timing
- Export/import pipeline configurations

Usage
-----

The GUI can be launched via the Pipeline Designer tool:

.. code-block:: bash

    python pipeline_designer.py

This will open the graphical interface for pipeline design.
